Collate

This is a demonstration of collate, a javascript and css implementation of paginated columns on the web. In order to use this yourself, just use the code in these two files, along with the jquery and jquery-scroll-to and modernizr dependencies. All you have to do is add the “collate” class to a div and the text inside will be put into columns and broken into pages based on the size of your browser window. For now, you also need to add the divs with the id's 'pageup' and 'pagedn' that are in this file to the bottom of your document, but that will go away in a future release

When this was first introduced, I wrote a long post about the motivations behind this. You can see that here You are looking at the result of this code. Columns, broken into pages, with scrolling assisted by javascript. Use the spacebar/shift-space or j/k to activate scrolling. There are little instructions to guide you to with scrolling they will only be shown you once, it uses localStorage to do so.

Please use github to track issues.
